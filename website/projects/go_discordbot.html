DiscordBot in Go

A Discordbot written in Go and with a "large" Backend

Features

  • Send a random Post from a subreddit to any Channel
  • Send a random Post from any subreddot via Dms
  • Play youtube videos in your voice Channel
  • Play some twitch streams in your voice Channel (Beta)
  • Customizable Prefix
  • Customizable commands

Backend

  • The Backend consisted of several Microservices:
    • User
    • Login
    • Server
    • Frontend
    • Statistics
    • Token-Updater
  • It had some Databases for different Purposes:
    • Users
    • Servers
    • Statistics

Technologies used

  • Docker and Docker swarm
  • Nodejs
  • Golang
  • MySql
  • MongoDB
  • With the Microservice Architecture

Statistics

  • Peak Users: ~75.000
  • Peak Servers: 85
  • Total Servers joined: ~700
