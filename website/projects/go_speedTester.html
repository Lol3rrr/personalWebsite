Website SpeedTester in Go

A Speed-testing tool written in Go

Features

  • Measures the response time for a specific page
  • Can send a varying amount of requests

Use Cases

  • Was originally developed to test real world performance of the C-Webserver
  • Good to test the raw speed of the Backend serving different files
