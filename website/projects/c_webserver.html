Webserver in C

A HTTP-Server completly written in ANSI-C

Features

  • Responds to GET requests
  • Port can be choosen before starting the Program
  • Automatically adds the '.html' extension if no extension is given
  • Automatically uses the index.html file in the folder if path ends in '/'
  • Has optional Debug output in case of a problem
  • Is one of the fastest web servers out there, speed test results can be found on the Github page

Plans

  • Add more configuration options
  • Implement the gzip or deflate Compression
  • Implement a local Cache for files to improve speeds
  • Create some sort of documentation for it

In short: Implement the whole HTTP/1.1 Standard and some extras for performance

Methods used

  • Structs
  • Single Linked lists
  • Extensive memory managment (malloc/free)
