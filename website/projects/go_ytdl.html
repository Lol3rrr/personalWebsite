Youtube Downloader in Go

A Youtube Downloader written in Go

Features

  • Builds on top of the 'rylio/ytdl' go package
  • Can download, basically any, video's audio from youtube
  • Can save the data to a file or keep it in memory
  • Allows you to download videos in parts

Use Cases

  • Was originally developed to load the Audio for my discord-bot
  • Can be used in any application that needs audio data from youtube

Plans

  • Expose more options to allow for different data types or resolutions
  • Add a cli for easy use
