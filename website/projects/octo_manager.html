Octo-Manager

A simple tool to automate some server related tasks

Features

  • Create backups
  • Restore backups
  • Run commands on a server
  • Interact with docker
  • Integrate with different services
  • Easily add new functionality
  • Easily add new modules to it
  • Easy to use in a CI/CD environment

Structure

  • Pretty flexible
  • Each feature belongs to a module
  • Easily add new modules and features
  • Jobs allow for multiple actions to be run, after one another
